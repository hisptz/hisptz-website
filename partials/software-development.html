Software development

Software development and integration is a continuous activity that is handled by the UDSM/UiO-MOHSW team. The UDSM/UiO-MOHSW team is responsible for
development and customization of the district based National Data Warehouse on the DHIS2 platform. The team is involved with

Design

Database design for 'rapid rollout' based on existing, but harmonized, data collection tools.

Maintenance

Software maintenance and data warehouse design based on revised tools.

Integration

DHIS2 integration with other related applications (e.g. LGMD).

Customization

Web portals and HTML reports customization for easy to access and encouraging information use
